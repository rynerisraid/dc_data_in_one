# dc\_data\_in\_one

项目用于总结日常的数据开发工作，以下三个模块组成：

* [ ] 用户系统
    * [ ] 注册  
    * [ ] 登录
    * [x] 验证
    * [x] 用户组
    * [x] 权限
* [ ] 数据血缘
    * [ ] SQL parser
    * [ ] SQL 脚本保存
    * [ ] 表级别数据血缘
    * [ ] 字段级别数据血缘
* [ ] 数据调度




## 技术栈

后端框架：Express + Sequelize + Sqlite + PostgreSQL\*
前端框架：Vue3.0
